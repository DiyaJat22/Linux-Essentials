
## Viewing the Public IP Address

Several websites allow you to view your public IP:

### icanhazip.com  
  Show Public IP:

```
curl icanhazip.com
```

### ifconfig.me 
  Show Public IP:

```
curl ifconfig.me
```

  Show All Network Details:

```
curl ifconfig.me/all
```

  Show User Agent:

```
curl ifconfig.me/ua
```

### ipinfo.io  
  Show Public IP:

```
curl ipinfo.io/ip
```

  Show Public IP with Location Details:

```
curl ipinfo.io
```
